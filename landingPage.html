About Me

After getting my Bachelors in Business Administration, I became an account manager for a digital marketing agency working with small business owners to create local SEO campaigns. As time went on, I found myself hanging around the web development team more and more, soaking up any bit of technical knowledge they could give me. My innate fascination with programming and being a creator is what ultimately led me to pursue a career in web development.

My strengths include my background in client-facing roles, learning new languages and tools quickly, responsive web design, Object Oriented Programming, static web app and dynamic web app development. I have experience using Java, Javascript, HTML, CSS, Bootstrap, PHP, MySQL, and GitHub to create websites and web applications.
